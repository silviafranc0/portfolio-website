# Personal Portfolio Website

## Overview

This project is a responsive personal portfolio website built using HTML and CSS. It showcases my background, technical skills, and selected projects as I work toward a career in cybersecurity, specifically in Blue Team analysis.

## Features

* Responsive design using a mobile-first approach
* Sticky footer layout using Flexbox
* Multi-page structure (Home and Projects)
* Project showcase with consistent card layout
* Contact form for direct communication
* Social media links (LinkedIn, GitHub)

## Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* Bootstrap Icons
* Google Fonts (Montserrat)

## Pages

* **Home (index.html)**
  Includes hero section, about section, and contact form.

* **Projects (projects.html)**
  Displays a collection of projects using a responsive layout:

  * Single-column layout on mobile
  * Multi-column grid layout on larger screens

## Responsive Design

The site follows a mobile-first design strategy:

* Content stacks vertically on small screens
* Expands into multi-column layouts on tablets and desktops
* No horizontal scrolling
* All text remains readable without zooming

## How to Run

1. Download or clone the repository
2. Open `index.html` in a web browser
